Perry Pakull, Trivadis AG

Prerequisites
--------------------------------------------------------------------------------
1. mig_home
   Variable "mig_home" must be defined in the system environment settings.
   Otherwise the scripts will not run properly.

2. Add %mig_home%\bin to the system path
